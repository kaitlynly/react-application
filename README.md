# React Static Story Page

This project is a React application that renders a static story page using reusable components.

## Story
The Tortoise and the Hare

## Project Structure

src/
- components/
  - Header.js
  - Content.js

- main.js

## How to Run

1. Clone the repository
2. Run npm install
3. Run npm run dev
